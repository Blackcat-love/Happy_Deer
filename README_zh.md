## 关于
该项目的主要功能是记录单个任务的执行时间，显示间隔时间，并最终分析数据以形成统计表。


## 主要功能

- 用户只需点击一个按钮即可完成记录，无需其他操作
- 分析数据并编制统计，形成每周、每月和每年的报告
- 在主页上清晰地显示记录状态，以保持合理的规律习惯
- 丰富的模式和主题以应对不同情况
